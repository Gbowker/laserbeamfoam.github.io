## timeVsLaserPower0

Defines the power of the incedent laser beam incident on the substrate in Watts with time in seconds. 

Example from tutorial `Plate2D`:
```cpp
(
    (0           4000)
    (1.999e-3    4000)
    (2e-3        1000)
    (4e-3        1000)
    (4.001e-3    4000)
    (1           4000)
)
```
