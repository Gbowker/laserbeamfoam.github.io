Disclamer

This software is designed to serve as a tool for automating certain aspects of the L-PBF simulation workflow. It can be used to generate input files along with .stl files needed for the LIGGGHTS discrete element method software. This application can also be used to generate input files related to laser position and properties required for running OpenFOAM simulations. It is important to note that this software is built as a helper tool for generating the simulation files, so it should work "out of the box" for most reasonable domain sizes and configurations, but some cases may need additional changes to input files from the user. The generated files are not guaranteed to be error-free, and the user is responsible for ensuring that the input files are correct and suitable for their specific simulation needs. The user assumes all risks associated with the use of this software and its output files.
